# ushiday marp

Marp = mdよりスライドを作成するプラグイン
## (02) 全スライドにスタイルを適用する方法

作成日 : 2020.07.08  **(ushiday)**

---
# 全スライドに対する共通設定をする場合

先頭部に次の項目を設定する事により、全スライドに共通設定が可能。

- `thema` テーマ設定
- `header` ヘッダー設定
- `footer` フッター設定

---
# 記述例

```md
---
marp: true
thema: gaia
header: "**ushiday** __Marp samples__"
footer: "by @ushiday (Chubu System.,Ltd.)"
---
```
